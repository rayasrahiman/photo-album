# photo-album
includes all albums and photos.


# Steps:

git clone [Repo](https://github.com/rayasrahiman/photo-album.git)

## Installation

```bash
npm i --save
```
Before proceeding make sure you have Expo app installed in your device.

## Run

```bash
npm start
```
